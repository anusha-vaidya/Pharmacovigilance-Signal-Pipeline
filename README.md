Pharmacovigilance Signal Detection Pipeline
A lightweight automated pipeline that ingests FDA FAERS snapshots, analyzes drug–reaction patterns, and publishes a live HTML dashboard via GitHub Pages.

👉 View Live Dashboard

https://anusha-vaidya.github.io/Pharmacovigilance-Signal-Pipeline/

What This Project Does
- Loads daily FAERS snapshot CSVs
- Extracts top drugs and top adverse reactions
- Generates an HTML report using Python + Pandas
- Publishes automatically to GitHub Pages through Actions
